# Student Marks Analyzer (Python)

## Description

A simple Python CLI project that analyzes student marks and calculates:

* Total marks
* Average
* Grade

This project demonstrates Python fundamentals:

* Variables
* Data types
* Loops
* Conditionals
* Functions
* Input/Output

## How to run

```bash
python main.py
```

## Example

```
Enter student name: Ravi
Enter number of subjects: 3
Enter mark 1: 80
Enter mark 2: 70
Enter mark 3: 90
```
