# Python To-Do CLI App

A simple command-line To-Do application built with Python.

## Features

- Add tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Edit task title
- Clear completed tasks
- Task priority (Low / Medium / High)
- Persistent storage using JSON

## Technologies

- Python
- JSON
